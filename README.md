# nostr-ratatui

`ratatui` で動く、Vim風キーバインドのNostr TUIクライアントです。

対応範囲:

- NIP-01: リレー接続、イベント購読、テキストノート、プロフィール
- NIP-02: フォローリストを使ったフォロータイムライン
- NIP-05: プロフィールのDNS識別子をHTTPSで検証（`✓` は検証済み、`?` は未検証）
- NIP-25: `+` / `-` / 絵文字リアクションの送受信
- NIP-30: Kitty Graphics Protocolによるカスタム絵文字の表示
- NIP-19: `npub` / `note` のBech32表示
- NIP-08: 従来の `#[index]` メンション送受信（入力時は `@npub...` / `nostr:note...`）
- NIP-18: リポスト（kind 6 / 16）の送受信
- Kitty Graphics Protocol: 対応端末でプロフィールのアイコン画像を表示

## 起動

```sh
nix develop
cargo run
```

投稿する場合は秘密鍵を環境変数で渡します。秘密鍵はファイルへ保存しません。

```sh
NOSTR_SECRET_KEY=nsec1... cargo run
```

未指定時は閲覧専用です。リレーは複数指定できます。

```sh
cargo run -- --relay wss://relay.example.com --relay wss://relay2.example.com
```

## キー操作

| キー | 操作 |
| --- | --- |
| `Tab` / `Shift-Tab` / `[` / `]` | フォロー / 全体タイムラインを切り替え |
| `1` / `2` | フォロー / 全体タイムラインへ移動 |
| `j` / `k` | 下 / 上へ移動 |
| `g` / `G` | 先頭へ移動してLIVE再開 / 末尾へ移動 |
| `l` / `Enter` | 詳細を開く |
| `h` / `Esc` | 詳細を閉じる |
| `m` | 設定を開く |
| `i` / `o` | 新規投稿 |
| `r` | 選択ノートへ返信 |
| `+` / `-` | Like / Dislike |
| `e` | 絵文字リアクション入力 |
| `R` | リポスト |
| `Ctrl-S` | 入力を送信 |
| `Esc` | 入力を破棄 |
| `q` | 終了 |

TLの表示領域が先頭にある間は、投稿の選択中や詳細表示中も含めてLIVE状態で新着へ追従します。`j` / `G`で画面が下へスクロールするとPAUSEDになり、選択中の投稿を維持したまま新着を上へ蓄積します。`k`で表示領域を先頭まで戻すか、`g`で先頭へ移動するとLIVEへ復帰します。

秘密鍵を指定した場合はフォロータイムラインから開始し、NIP-02の最新フォローリスト（自分自身を含む）を購読します。全体タイムラインではリレーから受信した全ユーザーの投稿を表示します。閲覧専用ではフォローリストを特定できないため、全体タイムラインから開始します。タブごとに選択位置とLIVE/PAUSED状態を保持します。

Kitty Graphics Protocolを検出できた場合は、タイムラインと詳細欄にプロフィールの`picture`を表示し、NIP-30の`emoji`タグに対応する本文中の`:shortcode:`を画像として描画します。PNG、JPEG、GIF、WebP、SVGに対応しています。画像取得はHTTPSのみに限定し、同時取得数、ダウンロード量、デコードサイズ、LRUキャッシュ件数に上限を設けています。画像の取得中・取得失敗時や非対応端末ではshortcodeをテキスト表示します。

本文中の `nostr:npub1...` / `nostr:nprofile1...` はプロフィール名付きのメンションとして表示します。`nostr:note1...` / `nostr:nevent1...` は参照イベントを取得し、引用ノートとして本文の下に表示します。

NIP-10のreplyは返信先イベントを取得し、返信先の投稿者と本文をタイムラインおよび詳細欄に表示します。旧形式のmarkerなし`e`タグにも対応します。

NIP-05のルート識別子 `_@domain` は、検証時には `_` を名前として使用し、画面上では `domain` のみ表示します。

> [!IMPORTANT]
> NIP-08は廃止済み仕様ですが、指定された既存イベントとの互換性のため対応しています。投稿時の `@npub...` / `nostr:npub...` / `nostr:note...` は `#[index]` と対応タグへ変換されます。
